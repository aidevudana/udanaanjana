# Udana Anjana — CV Website

Personal portfolio and CV website for Udana Anjana, Social Media Marketing Manager & Creative Professional based in Colombo, Sri Lanka.

## Features
- Responsive single-page design
- Skills showcase with 9 specialty areas
- Services listing
- About section with tool proficiency bars
- Contact section

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `udana-anjana-cv`)
2. Upload the `index.html` file to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save** — your site will be live at `https://yourusername.github.io/udana-anjana-cv`

## Customise
- Update your email, phone, and LinkedIn URL in the Contact section
- Replace placeholder contact details with your real info
